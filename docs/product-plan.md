# Produktplan

## Kjerneverdi

Better Discounts skal løse et konkret Shopify-problem: rabatten finnes i checkout eller på produktsiden etter spesialtilpasning, men vises ikke nødvendigvis tydelig i butikkfronten som standard.

Appen skal gjøre dette enkelt:

- Installer app.
- Opprett rabatt.
- Vis rabatt automatisk på frontend.

## Målgruppe

- Små og mellomstore Shopify-butikker som ikke vil redigere theme-kode.
- Byråer som vil gi kundene en rask rabattvisning uten spesialprosjekt.
- Merchants som vil ha enkel abonnementsmodell og rask time-to-value.

## MVP

- Embedded admin med enkel opprettelse av rabattkampanje.
- Produktgrense per abonnement.
- Frontend-badge via theme app extension.
- Planer: `Free`, `Plus`, `Business`.
- Enkel onboarding som peker kunden til theme editor med deep link.

## Prisforslag

- `Free`: 10 produkter, gratis.
- `Plus`: 50 produkter, foreslått `$19 / month`.
- `Business`: ubegrenset, foreslått `$79 / month`.

Prisene er foreløpige antakelser og bør valideres mot konkurrenter før lansering.
